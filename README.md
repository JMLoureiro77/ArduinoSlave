# ArduinoSlave

Arduino library to Control Arduino by Serial with ESP8266 or other

To upgrade esp8266 with:

- 8 analog reads 0-5v
- 11 digital 5v IOs
- PWM 
- Variable exchange 
- ESP unsupported libraries can be put on Slave & send values
